# Taller React

`social-taws` contains the web fontend codebase.

## Setting up

1. Clone

`git clone https://github.com/Taws-Espol/front-react.git`

2. Change directory

`cd social-taws`

3. Install dependencies

`npm install`

4. Run

`npm start`
